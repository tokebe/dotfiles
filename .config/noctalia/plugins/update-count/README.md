# Update Count Plugin

A widget that periodically scans for new updates and notifies you of them.

## Features

- **Plugin Settings**: Highly customizable through its settings
- **Custom Icon**: Icon is able to be set by you through the plugin settings
- **Update Checking**: You are able to customize how often it checks for updates
- **Hidden Widget**: When there are no updates, the widget is able to be completely hidden from your bar

## Usage

The widget displays an Icon next to a number (How many updates are pending):
- **Click!**: One simple click will open a terminal and start updating your system
- **Hover!**: For a tooltip telling you how many updates are pending

## Requirements

- Noctalia Shell 3.6.0 or higher
- [Gum](https://github.com/charmbracelet/gum)
- Update scripts and checking only support Arch (pacman, yay, paru), Fedora (dnf)
